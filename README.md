# Flashcard Quiz Web

Web hỗ trợ học từ vựng/kiến thức bằng flashcard và quiz để ôn tập, ghi nhớ hiệu quả.

## Mục tiêu MVP
Sản phẩm cần chạy được các chức năng cơ bản:
- Xem danh sách bộ flashcard
- Tạo bộ flashcard mới
- Thêm flashcard vào bộ
- Học bằng cách lật flashcard
- Làm quiz từ flashcard
- Xem kết quả đúng/sai

## Cấu trúc file
- `index.html`: giao diện chính
- `style.css`: giao diện CSS
- `script.js`: phần xử lý chung
- `storage.js`: lưu và đọc dữ liệu bằng localStorage
- `flashcard.js`: chức năng flashcard
- `quiz.js`: chức năng quiz

## Phân công
- Người 1:
  - `index.html`
  - `style.css`

- Người 2:
  - `storage.js`
  - `flashcard.js`

- Người 3:
  - `quiz.js`
  - Slide báo cáo giữa kỳ

## Luồng demo
Danh sách bộ thẻ → tạo bộ thẻ → thêm flashcard → học flashcard → làm quiz → xem kết quả
